## Statistics

This **security-txt-registry** server:

* Monitors **9,312 domains** for `security.txt` documents & signatures.
* Contains **1,623 documents** with valid `security.txt` contents.
* Contains **32 GPG signatures** for `security.txt` documents.
* Processed **279,621,452 objects** during its lifetime.
* Crawled **85,948,708 URLs** on the Interwebs.
* Handled **74,929 `whois` connections** via IPv4/IPv6.
* Handled **13,842 API requests** via HTTPS.
